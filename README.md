# Backend

## Crear entorno virtual
```
python -m venv venv
```

## Ingresar entorno virtual
```
venv\Scripts\activate
```

## Instalar dependencias
```
pip install -r requirements.txt
```

## Aplicación en debug off
```
flask run
```

## Aplicación en debug on
```
python app.py
```

## Salir del entorno virutal
```
deactivate
```
